## Video Lesson

### Backend

#### Run

    $ yarn start

#### Run dev

    $ yarn run dev

### Frontend

#### Run

    $ cd frontend

    $ npm uninstall -g react-native-cli

    $ Add "server: {port: 8088}" in file metro.config.js

    $ px react-native run-android --port 8088

    $ npx react-native start

### Frontend_expo (Expo)

#### Run

    $ cd frontend

    $ yarn start

#### Build file apk

    $ cd frontend

    $ expo build:android -t apk

npx react-native run-android --port 8088

npm uninstall -g react-native-cli
